# Espiritu Emprendedor

A simple Angular application for the "Espiritu Emprendedor" project.

## Prerequisites

- Node.js (recommended version 18 or newer)
- npm (included with Node.js)

## Installation

Install dependencies from the project root:

```bash
npm install
```

## Development server

Run the local development server:

```bash
npm start
```

Then open your browser at `http://localhost:4200`. The app will reload automatically when you change files.

## Build

To build the project for production:

```bash
npm run build
```

The build output will be written to the `dist/` folder.

## Lint

To run the linter:

```bash
npm run lint
```

## Notes

- This project is built with Angular 17.
- If you need to run a specific Angular CLI command, use `npx ng <command>`.
